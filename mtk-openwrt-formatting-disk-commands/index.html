安装工具包

安装两个工具包

该工具可通过SDK的make menuconfig选上并编译生成

分区

fdisk /dev/sda进入交互模式

  • n add a new partition: 基本上一路默认即可，选择primary，选择容量等。

  • t change a partition type: 修改新分区的类型，选择83 Linux，可通过L查看支持的类型。

  • w write table to disk and exit: 保存退出
