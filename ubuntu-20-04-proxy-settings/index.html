系统代理

chrome浏览网页需要设置系统代理。

GUI方式修改

依次选择：Settings -> Network -> Network Proxy -> Manual

修改如下：

  • HTTP、HTTPS和FTP：这里不用填，否则会无法访问网页。
  • Socks：这里必填。
  • Ignore：这里不能出现127.0.0.1或者127.0.0.0/8，否则也无法访问网页。
