获取寄存器的值

物理地址(0x10110080) - 物理BASE地址(0x10000000) = 偏移量(0x110080)
虚拟BASE地址(0xb0000000) + 偏移量(0x110080) = 虚拟地址(0xb0110080)
因为reg命令只需要一个偏移量就够了，所以没用到虚拟地址。

得到的值再进行与操作，就可以拿到对应端口的Link Status了。
